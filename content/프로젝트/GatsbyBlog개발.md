### 구현한 기능

- Graphql를 활용한 폴더 구조 파악으로 자동화되는 목차
- 등록된 게시글 수와 현재 페이지 번호에 반영된 페이지네이션
- 라이트모드와 다크모드 기능
- 반응형 레이아웃
- 게시글을 자세히 볼수 있는 페이지
- 목차와 게시글을 한눈에 볼수있는 메인 페이지
